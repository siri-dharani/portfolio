Bomb Defusal

A simple web-based game where players must type "defuse" click enter before the 10-second countdown hits zero. If they fail, the bomb explodes with a BOOM!! message. Built using HTML, CSS, and JavaScript.


GuessNumber

A simple number guessing game where players must guess a randomly generated number between 1 and 100. The game provides hints like "Too High" or "Too Low" until the correct number is found. Built using HTML, CSS, and JavaScript.


Counter App

A simple counter application with Increase, Decrease, and Reset functionalities. The counter changes color based on its value: green for positive, red for negative, and black for zero. Built using HTML, CSS, and JavaScript.


Food Munch

A simple, mobile-friendly food delivery website with a navbar, banner, and call-to-action buttons. Built using HTML, CSS, and Bootstrap for a clean and responsive design.


Light&Cat

A fun interactive webpage where clicking ON/OFF switches a bulb's state and changes a cat's image. Built with HTML, CSS, JavaScript, and Bootstrap for dynamic effects.


Save Pad

A simple notepad that lets you type and save text in your browser using local storage. Your text remains even after refreshing the page, making it a quick and convenient tool for notes.
